# langchain-gemini-starter

LangChain + Gemini Starter

A simple demo project showing how to use LangChain with Google Gemini to build prompt-based LLM workflows.

What It Does

Connects to Gemini (gemini-2.0-flash)

Creates parameterized prompts

Builds runnable chains (Prompt → LLM → Output Parser)

Demonstrates basic structured AI responses

Requirements

Python 3.10+

Google Cloud project with Gemini API enabled

Billing enabled (required for API access)

Purpose

This project is for learning and experimenting with modern LLM workflows and serves as a foundation for building agentic AI systems.
